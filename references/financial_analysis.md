# 财务分析框架

本框架配合 `stock_data.py` 脚本获取的财务数据，用于指导公司财务分析的方法论。

## 核心财务指标分析

### 1. 盈利能力指标

**数据来源**: `financial_data_handler` 返回的近12期核心财务指标

**分析维度**:
- **毛利率**: 评估产品定价能力和成本控制水平，高于行业平均显示竞争优势
- **净利率**: 综合反映费用控制能力和盈利质量，关注非经常性损益影响
- **ROE（净资产收益率）**: 衡量股东回报效率，持续高于15%为优秀水平
- **ROA（总资产收益率）**: 评估资产使用效率，区分轻资产与重资产模式特征

**分析方法**:
- 12期趋势变化：识别改善或恶化方向
- 同期对比：同比增长/下降幅度
- 行业基准：与同行业公司对比
- 杜邦分析：将ROE分解为净利率、总资产周转率、权益乘数

### 2. 成长性指标

**数据来源**: 近12期营收、净利润、每股收益数据

**分析维度**:
- **营收增长率**: 判断业务扩张速度和质量（内生 vs 外延）
- **净利润增长率**: 评估盈利增长可持续性，关注扣非净利润
- **每股收益增长率**: 反映股东回报增长，考虑股本扩张影响

**分析方法**:
- 连续3-5年趋势判断
- 与行业增速对比
- 增长驱动因素分析
- 季节性波动规律

### 3. 偿债能力指标

**数据来源**: 资产负债相关数据

**分析维度**:
- **资产负债率**: 评估财务杠杆水平，关注行业特性差异
  - 制造业：40-60%为合理区间
  - 房地产：70-80%属正常范围
  - 金融业：90%以上
- **流动比率/速动比率**: 衡量短期偿债能力，关注流动性风险
- **利息保障倍数**: 评估利息支付能力

**分析方法**:
- 债务结构分析（长短期债务占比）
- 债务成本变化趋势
- 与现金流匹配度评估

### 4. 运营效率指标

**数据来源**: 存货、应收账款、总资产周转数据

**分析维度**:
- **应收账款周转率**: 反映回款速度和客户质量，周转天数持续增加需警惕
- **存货周转率**: 评估库存管理效率和产品适销性
- **总资产周转率**: 衡量资产使用效率，识别商业模式特征

**分析方法**:
- 周转天数变化趋势
- 与行业平均水平对比
- 季节性波动分析

### 5. 现金流量指标

**分析维度**:
- **经营性现金流**: 与净利润对比评估盈利质量，现金流应大于净利润
- **自由现金流**: 反映真正可支配现金，是分红和投资能力的核心指标
- **现金流覆盖率**: 经营性现金流/净利润，低于1需关注应计项目

**分析方法**:
- 现金流与净利润匹配度分析
- 季节性波动规律
- 资本支出合理性评估

## 估值分析

**数据来源**: `financial_data_handler` 返回的近一年估值统计

### 相对估值法

**分析维度**:
- **PE（市盈率）**: 适合盈利稳定公司，结合历史PE区间和行业平均PE评估
- **PB（市净率）**: 适用于重资产、周期性行业和金融机构
- **PS（市销率）**: 适用于成长初期或尚未盈利企业
- **PEG（市盈率相对盈利增长比率）**: PEG<1可能低估，PEG>1可能高估

**分析方法**:
- 当前值与历史均值对比
- 历史分位数位置判断（25%、50%、75%分位数）
- 行业横向对比
- 增长预期调整

### 估值判断标准

- **历史分位数**: 25%以下为低估区间，75%以上为高估区间
- **相对估值**: 与行业平均对比，低于行业平均-20%为明显低估
- **安全边际**: 结合基本面质量评估估值合理性

## 财务分析流程

### 第一步：数据质量检查
1. 检查财务数据的完整性和连续性
2. 识别异常值或重大会计政策变更
3. 确认数据时效性

### 第二步：趋势分析
1. 计算各指标的同比/环比变化
2. 识别改善或恶化趋势
3. 分析驱动因素和可持续性

### 第三步：质量评估
1. 盈利质量（现金流匹配度）
2. 资产质量（减值准备、周转率）
3. 债务质量（期限结构、利率）

### 第四步：同业对比
1. 与行业龙头对比
2. 与行业平均水平对比
3. 识别相对优势和劣势

### 第五步：风险识别
1. 财务杠杆风险
2. 流动性风险
3. 盈利波动风险
4. 资产质量风险

## 行业特定分析要点

### 制造业
- 关注产能利用率变化
- 分析原材料成本波动传导
- 评估技术升级和资本支出需求

### 消费品行业
- 品牌价值及渠道控制能力
- 新品成功率和库存周转
- 毛利率稳定性分析

### 科技行业
- 研发投入效率（ROI）
- 用户增长和变现能力
- 平台网络效应评估

### 金融行业
- 资产质量（不良率、拨备覆盖率）
- 资本充足率变化
- 净息差及手续费收入占比

## 红色预警信号

### 财务风险信号
1. 收入与经营性现金流长期背离
2. 应收账款异常增长且周转率下降
3. 存货周转率持续恶化
4. 毛利率持续下滑且无法解释
5. 负债率快速上升且现金流紧张

### 经营风险信号
1. 核心管理人员频繁变动
2. 重大关联交易且缺乏合理解释
3. 频繁变更会计师事务所
4. 持续大额对外投资且收益不明

## 报告撰写要点

### 财务摘要部分
- 突出关键指标的变化趋势
- 使用具体数据支持分析结论
- 强调财务表现的可持续性

### 风险提示部分
- 基于数据量化风险程度
- 区分短期波动与长期风险
- 提供明确的关注要点

### 投资建议部分
- 基于估值分析和基本面质量
- 明确投资逻辑和假设条件
- 区分适合的投资者类型
